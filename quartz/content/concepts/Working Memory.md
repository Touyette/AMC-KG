# Working Memory

> The severely capacity-limited active workspace of cognition — roughly 7 ± 2 items — that constrains what can be simultaneously manipulated; what gets selected for working memory IS relevance realization happening.

## Vervaeke's framing
Working memory (Baddeley's model: phonological loop, visuospatial sketchpad, central executive) is the bottleneck of conscious cognition. Its severe capacity limit (~7 ± 2 chunks, and recent estimates suggest even fewer independent items) means that only an infinitesimal fraction of potentially relevant information can be held in mind at once. Every inference, every problem-solving step, every deliberate action requires information to be active in working memory — and so the selection of what enters it is the most critical cognitive operation there is (Ep 28).

**Working memory and the frame problem**: Vervaeke's key argument: the question of what gets loaded into working memory IS the frame problem at the individual cognitive level. Some mechanism must already determine relevance *before* the content can be processed in working memory — because working memory is too small to load everything and then sort it out. That pre-conscious relevance selection mechanism is RR. Working memory is where the conscious products of RR appear; RR itself operates largely below the level of working memory (Ep 28).

**Working memory and insight**: In insight problem solving, the phenomenology of the "aha" moment often corresponds to a restructuring of what is held in working memory — the impasse involves being stuck with an unhelpful framing loaded in working memory, and the insight involves releasing that framing and loading a new one. Attentional scaling (breaking inappropriate frame → moving through → new frame) can be understood as a working memory event: releasing the grip of the current contents and allowing a reorganized set to emerge (Ep 28).

**Implication for the meaning crisis**: One reason the meaning crisis is not solvable by mere information provision or argument: the arguments have to get through working memory and connect to the agent's actual salience landscape — which is shaped by the RR machinery, not by what is in working memory. Telling someone why life is meaningful doesn't restructure their relevance realization.
