# Enactive Demonstrative Reference

> Vervaeke's term for the non-linguistic, enacted salience-tagging of "this one here now" that underlies all categorization and representation — the participatory-enactive foundation of demonstrative reference (pointing, indexicals), grounded in FINSTING.

## Vervaeke's framing
Demonstrative reference in language — "this," "that," "here," "now" — presupposes a prior cognitive act: the picking-out of a particular as "this one" from an undifferentiated field. Vervaeke's term "enactive demonstrative reference" names this prior act at the pre-linguistic, participatory level. It is "enactive" because it is not a passive labeling but an active engagement with the environment that singles out what is here and now; it is "demonstrative" because it is an act of pointing (in the broad cognitive sense) before it is an act of categorizing (Ep 29).

**Grounded in FINSTING**: [[FINSTING]] (Pylyshyn's finding that the mind tracks individual objects pre-categorically) provides the empirical evidence that enactive demonstrative reference is real and cognitively prior to propositional knowledge. The mind can track "that one" without yet knowing "what it is." Vervaeke uses this to argue that the most basic cognitive relation to the world is participatory contact, not representational encoding (Ep 29).

**Sub-semantic level**: Enactive demonstrative reference operates below the semantic level — below meaning in the propositional sense. It is the level at which the world is differentiated into "this" and "not-this" before any content is assigned to either. This sub-semantic layer is where Relevance Realization begins: what shows up as "this" (salient, trackable, here) is already an expression of the RR machinery at work (Ep 29).

**Implication for the hierarchy of knowing**: Because enactive demonstrative reference operates at the participatory level (enacted contact without representation), it supports the hierarchical dependency argument: propositional knowing depends on demonstrative reference, which depends on participatory engagement. You cannot begin to know "that" propositionally without first having established "this" participatorily (Ep 29).
