# Parasitic Processing

> A self-reinforcing cognitive-existential spiral in which the very adaptive machinery that makes an agent intelligent turns against the agent, consuming resources and agency without producing genuine meaning.

## Vervaeke's framing
The machinery that makes cognition adaptive — heuristics (representativeness, availability), self-organization, encoding specificity, confirmation bias, the tendency to make events salient and track patterns — cannot be switched off. That same machinery, triggered by a bad event, can start a negative feedback loop: the bad event becomes highly salient → the availability heuristic inflates its probability → encoding specificity makes negative memories more accessible → confirmation bias reinforces the gloomy picture → anxiety rises → cognitive flexibility narrows → problem-solving capacity drops → more failures → increasing anxiety → fatalism. The system re-organizes around every attempted intervention, making wilful self-correction nearly impossible. Vervaeke (with Leo Ferraro) named this **Parasitic Processing** because it lives inside the adaptive machinery — like a parasite it takes up life within you while taking life *from* you (Ep 13).

Parasitic Processing is Vervaeke's cognitive-scientific gloss on **Dukkha**: all of life is *threatened by* this spiral, not constituted by pain. It is also the internal correlate of Marc Lewis's **Reciprocal Narrowing** — these two spirals interpenetrate (loss of cognitive flexibility = loss of world-options), reinforcing each other into full entrapment. The appropriate response is not propositional self-correction but a **counteractive dynamical system** (the Buddha's Eightfold Path) that operates simultaneously at the level of cognition, character, and consciousness (Ep 13).
