# Practices Index

_Controlled vocabulary for practices in AMC — psycho-technologies Vervaeke discusses as meaningful transformative tools. One row = one canonical note._

| Canonical Name | Aliases | Tradition | Notes |
|---|---|---|---|
| Mindfulness | sati, sati practice, mindfulness meditation | Buddhism | Sati: present-moment non-judgmental awareness; counteracts parasitic processing and modal confusion |
| Vipassana | vipassana, insight meditation | Buddhism | Insight meditation; direct seeing of impermanence and non-self |
| Metta | metta, lovingkindness meditation | Buddhism | Cultivation of agapic goodwill toward all beings; counteracts alienation |
| Eightfold Path | Eightfold Path, Noble Eightfold Path | Buddhism | Buddhist psycho-technology across eight dimensions: right view → right concentration |
| Contemplation | contemplation, contemplative practice, theoria | Multiple | Sustained non-discursive attending; distinct from discursive meditation; recurs across Platonic, Christian, and Buddhist traditions |
| Lectio Divina | lectio divina, sacred reading | Christianity (monastic) | Slow recursive engagement with text enacting inner dialogue; model for how symbol works transformatively |
| Prosoche | prosoche, self-attention | Stoicism | Stoic practice of moment-to-moment attention to one's inner state; Hadot; basis for Marcus Aurelius's Meditations |
| Premeditatio | premeditatio mortis, premeditation of death | Stoicism | Deliberate rehearsal of loss and death to cultivate equanimity and proper valuing |
| View from Above | view from above, Stoic visualization | Stoicism | Visualization practice: take the perspective of a cosmos-wide observer to achieve detachment from local concerns |
| Objective Seeing | objective seeing, Hadot's objective seeing | Stoicism / Hadot | Seeing things as they really are, stripped of the projections of need and desire |
| Spiritual Exercises | spiritual exercises, Hadot's spiritual exercises | Multiple (Hadot) | Hadot: philosophy practiced as a set of daily exercises for transforming the self; not merely theoretical |
| Socratic Dialectic | dialectic as practice, elenchus | Platonism | Cross-examination aimed at aporia; trains second-order thinking; models epistemic humility |
| Internalizing the Sage | internalizing the sage, inner dialogue | Stoicism / Multiple | Taking on the normative perspective of a wise exemplar (Socrates, the Stoic Sage) until it constitutes one's metacognitive voice |
| Flow State Cultivation | cultivating flow, flow practice | Multiple / Csikszentmihalyi | Deliberately engineering the conditions (skill/challenge match, tight feedback) for flow; trains insight cascade and implicit learning |
| Active Imagination | active imagination | Jungian | Jung: allowing images to self-organize without ego direction so unconscious archetypes can dialogue with the ego; imaginal not imaginary |
| Anagogic Ascent | anagogic ascent, anagoge as practice | Neoplatonism / Multiple | Moving through symbols in a participatory, transformative way toward fuller participation in being |
| Communitas | communitas, liminal community | Ritual Studies / Turner | Deliberate cultivation of role-dissolving community as transformative practice; Turner's liminality |
| Method of Loci | method of loci, memory palace | Ancient Rhetoric | Spatial memory technique; externalization of reasoning via place; precursor to modern psycho-technology thinking |
| Active Open-Mindedness | active open-mindedness, AOM | Contemporary / Baron | Cultivated disposition to seek disconfirming evidence; epistemic virtue and cognitive practice |
